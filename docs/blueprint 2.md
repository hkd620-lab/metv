# **App Name**: LexiList

## Core Features:

- Word Entry Form: Users can input an English word and its Korean definition into dedicated fields.
- Vocabulary List Display: Display a dynamic, scrollable list of all entered English words and their corresponding Korean definitions.
- Add Word to List: A button to add the currently entered word and definition to the vocabulary list.
- Delete Word from List: Ability to remove individual vocabulary entries from the displayed list.
- Automatic Definition Suggestion: An AI tool that suggests relevant Korean definitions or examples for an entered English word, based on common usage and context, before the user manually confirms.

## Style Guidelines:

- Primary color: A balanced, deep cerulean (#298ECC) to evoke focus and clarity in learning.
- Background color: A very light, muted blue (#EBF3F7) for a clean and comfortable reading experience.
- Accent color: A fresh, vibrant aqua-green (#44CEAF) to highlight interactive elements and draw attention to key actions.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean lines and excellent readability across all text sizes, maintaining a consistent and polished look.
- Utilize simple, clean line-art icons that complement the minimalistic aesthetic and provide clear visual cues for actions like 'add' and 'delete'.
- A clean, responsive layout designed with Tailwind CSS, featuring clear separation between the input form and the vocabulary list. Ample white space will ensure content is easily digestible.
- Subtle, fast transition animations for adding or removing items from the list, providing smooth user feedback without being distracting.
